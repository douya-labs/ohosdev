最快、最稳的联系方式是 **GitHub**：

- **Bug 报告 / 内容错误 / 想法** → [开 issue](https://github.com/douya-labs/harmony-app-dev/issues)
- **教程稿件、翻译、新增参考** → [提 PR](https://github.com/douya-labs/harmony-app-dev/pulls)
- **一般讨论** → 用 [Discussions](https://github.com/douya-labs/harmony-app-dev/discussions)

确实不适合 GitHub 的事项（安全披露、合作咨询、下架请求），请邮件：

> **hello [at] ohosdev.com**

我们是小团队，请给我们几个工作日回复。
