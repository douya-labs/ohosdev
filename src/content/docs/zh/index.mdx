## 这是什么？

**ohosdev** 是 [`harmony-app-dev`](https://github.com/douya-labs/harmony-app-dev) AgentSkill 的公开 Web 镜像 —— 一个手工整理、按能力域分类的 HarmonyOS / OpenHarmony 开发知识库。

<CardGrid>
  <Card title="阅读教程" icon="document">
    覆盖 ArkTS、ArkUI 动画、Canvas、Widget、HarmonyOS 与 Android 迁移等长文。写给真正在做产品的开发者。
  </Card>
  <Card title="速查参考" icon="open-book">
    38 篇按能力域分类的参考文档（UI / 动画 / 卡片 / 网络 / 存储 / 并发 ……），与上游 AgentSkill 保持一致并持续更新。
  </Card>
  <Card title="接入你的 AI" icon="rocket">
    将 `harmony-app-dev` AgentSkill 装进 Claude / Cursor / OpenClaw，可以避免 AI 编造不存在的鸿蒙 API。Web 和 Skill 共享同一份知识源。
  </Card>
  <Card title="开源 / 社区共建" icon="github">
    所有内容都在同一个 GitHub 仓库，欢迎提 PR：教程、翻译、新增能力参考都可以。
  </Card>
</CardGrid>
