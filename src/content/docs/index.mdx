## What is ohosdev?

**ohosdev** is the public web mirror of the [`harmony-app-dev`](https://github.com/douya-labs/harmony-app-dev) AgentSkill — a hand-curated, capability-classified knowledge base for building real apps on HarmonyOS and OpenHarmony.

Two ways to use it:

<CardGrid>
  <Card title="Read the tutorials" icon="document">
    SEO-friendly long-form articles on ArkTS, ArkUI animation, Canvas drawing, widgets, and HarmonyOS-vs-Android migration. Written for developers who actually ship.
  </Card>
  <Card title="Skim the reference" icon="open-book">
    38 capability-classified reference files (UI, animation, widget, network, persistence, concurrency, …) sourced from the upstream AgentSkill. Updated continuously.
  </Card>
  <Card title="Plug it into your AI" icon="rocket">
    Install the `harmony-app-dev` AgentSkill into Claude / Cursor / OpenClaw and your AI stops inventing fake HarmonyOS APIs. The website and the skill share the same source of truth.
  </Card>
  <Card title="Open source, community-driven" icon="github">
    Everything lives in one GitHub repo. PRs welcome — for tutorials, translations, or new capability references.
  </Card>
</CardGrid>
